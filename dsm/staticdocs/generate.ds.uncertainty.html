Generate data from a fitted detection function

Usage

generate.ds.uncertainty(ds.object)

Arguments

ds.object
a fitted detection function object (as returned by a call to ddf.ds().

Description

When ds.uncertainty is TRUE, this procedure generates data from the fitted detection function (assuming that it is correct).

Note

This function changes the random number generator seed. To avoid any potential side-effects, use something like: seed <- get(".Random.seed",envir=.GlobalEnv) ### Run some code assign(".Random.seed",seed,envir=.GlobalEnv)
