Generate a vector of residuals to be mapped back onto the data

Usage

generate.mb.sample(num.blocks.required, block.size, which.blocks, dsm.data, unit.info, 
  n.units)

Arguments

num.blocks.required
number of blocks that we need.
block.size
number of segments per block.
which.blocks
which blocks should be sampled.
dsm.data
the $data element of the result of a call to dsm.fit.
unit.info
result of calling block.info.per.su.
n.units
number of sampling units.

Value

vector of log-residuals

Description

Not usually used on its own, called from within dsm.var.movblk.
